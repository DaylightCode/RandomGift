# RandomGift

⚠️ **Commercial Project** - Source code is under NDA. This repository contains project overview and demo materials only.

Telegram Mini App with prize wheel, leaderboards, and lottery system featuring instant gift withdrawals.

## ⚙️Stack

**Frontend:**
* Next.js v15 (App Router + Server Actions)
* React v19
* TypeScript
* Tailwind CSS
* Framer Motion

**Backend & Database:**
* Prisma ORM v5.0
* PostgreSQL

**Telegram Integration:**
* Telegram Bot API
* Telegram Web Apps API

## ✨ Key Features

✅ **Admin Panel** - Create and manage giveaways with custom settings
✅ **Automated Prize Distribution** - Instant gift delivery upon winner selection
✅ **Real-time Participant Tracking** - Live updates of entries and statistics
✅ **Telegram Authentication** - Seamless user authorization via Telegram
✅ **Responsive Design** - Optimized for Telegram WebView on all devices
✅ **Adaptive Themes** - Auto-syncs with Telegram appearance
